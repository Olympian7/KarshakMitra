# **App Name**: Kerala Krishi Mitra

## Core Features:

- Personalized Farm Profile: Create a digital representation of the farm by inputting details such as land size and soil type; the AI tool will then provide insights using local soil and climate data.
- Conversational Assistant: Voice-enabled chatbot for farmers; it should understand both Malayalam and English. Translate the messages from Malayalam to English and vice versa on a single click.
- Activity Tracking: A digital diary that allows farmers to log farming-related events via voice; The AI tool will analyze these to identify opportunities for improvements.
- Proactive Alerts & Reminders: Send important actionable notifications to the farmer related to weather changes, pests and important farm related tasks
- Government Schemes Hub: Hub that simplifies details on the current existing government schemes and helps farmers to check for eligibility. The LLM may incorporate details from a website to enrich this data if necessary as a tool.
- Market Trends Tool: Market intelligence dashboard with details such as market price and ongoing trends to help farmers make smarter selling decisions. The LLM may consult external data to provide more relevant information, and so this serves as a tool.

## Style Guidelines:

- Primary color: Forest green (#388E3C). Evokes Kerala's lush landscape, while providing a vibrant yet calming feel.
- Background color: Light green (#E8F5E9). Keeps a consistent analogous scheme that contrasts well with the primary.
- Accent color: Golden yellow (#FDD835). Like the primary, draws on a natural referent; reminiscent of rice, spice, or sunlight. Its greater saturation gives sufficient contrast for highlighting.
- Font: 'PT Sans', a humanist sans-serif providing both a modern and warm feeling. The intention is that this should feel like a 'techy friend'.
- Use simple, clear icons related to farming, weather, and markets.
- Clean, intuitive layout that adapts to different screen sizes and devices.
- Subtle animations and transitions to provide a smooth and engaging user experience.
